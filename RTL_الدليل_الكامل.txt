7️⃣ أمثلة عملية
═══════════════════════════════════════════════════════════════

مثال 1: إضافة صفحة جديدة
────────────────────────────
<?php
require_once("top.php");
?>
<!DOCTYPE html>
<html lang="<?php echo $_SESSION['lang']; ?>" 
      dir="<?php echo ($_SESSION['lang'] == 'ar') ? 'rtl' : 'ltr'; ?>">
<head>
    <?php require_once("header.php");?>
</head>
<body>
    <!-- المحتوى هنا -->
</body>
</html>

مثال 2: إضافة عنصر جديد بـ CSS
────────────────────────────────
/* التصميم العادي */
.my-element {
    display: flex;
    text-align: left;
}

.my-element i {
    margin-right: 0.5rem;
}

/* RTL */
[dir="rtl"] .my-element {
    flex-direction: row-reverse;
    text-align: right;
}

[dir="rtl"] .my-element i {
    margin-right: 0;
    margin-left: 0.5rem;
}

═══════════════════════════════════════════════════════════════
🐛 حل المشاكل
═══════════════════════════════════════════════════════════════

المشكلة: العناصر لا تتحول لـ RTL
الحل: 
   1. امسح Cache (Ctrl + F5)
   2. تأكد من تحميل rtl-support.css
   3. تحقق من dir="rtl" في HTML

المشكلة: الأيقونات في المكان الخطأ
الحل:
   أضف flex-direction: row-reverse للعنصر

المشكلة: الأرقام معكوسة
الحل:
   أضف direction: ltr للعنصر الذي يحتوي الأرقام
