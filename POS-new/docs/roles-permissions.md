## الأدوار والصلاحيات (مبدئي)

الأدوار:

- كاشير (cashier)
- مدير (manager)
- مسؤول (admin)

الصلاحيات الأساسية:

- كاشير:

  - إنشاء فاتورة بيع POS
  - البحث عن الأصناف
  - إنشاء عميل سريع
  - طباعة الإيصال

- مدير:

  - كل صلاحيات الكاشير
  - CRUD أصناف/تصنيفات/وحدات
  - إدارة العملاء/الموردين
  - إنشاء مشتريات
  - تقارير أساسية

- مسؤول:
  - كل صلاحيات المدير
  - إعدادات النظام
  - إدارة الأدوار والصلاحيات
  - نسخ احتياطي واستعادة
